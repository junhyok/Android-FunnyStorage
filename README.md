# Android-FunnyStorage
웃음 저장소 앱

### [작품 소개]
웃긴 동영상(Youtube영상), 웃긴 이미지(img,gif)들을 나만의 공간에 저장해 두고 볼 수 있는 앱 입니다. <br/>
그리고 저장해 놓은 내용들을 다른 사람들에게 공유(카카오톡, 페이스북)할 수 있는 기능을 가지고 있습니다.
